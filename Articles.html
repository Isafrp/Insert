A Comprehensive and Configurable Simulation Environment to Support Vehicular Named Data Networks Applications

Authors: Guilherme Araújo, Maycon Peixoto, Leobino Sampaio

Date: November 2023

The Named Data Networking (NDN) architecture is highly suitable for vehicular applications and inter-vehicular communication (IVC). To facilitate experimentation with Vehicular Named-Data Networking, we introduce NDN4IVC, an open-source simulation framework that enables more realistic application testing in VANETs. This framework allows real-time bidirectional communication between SUMO and NS3, providing deeper insights into road traffic and vehicular mobility.

Type: Journal Article | Publication: Computer Networks, 235

Enhancing Network Slicing Architectures with Machine Learning, Security, Sustainability, and Experimental Network Integration

Authors: Multiple Researchers

Date: July 2023

Network Slicing (NS) is a key technique widely used in 5G computing strategies and Industrial IoT. The Slicing Future Internet Infrastructures (SFI2) architecture addresses the diversity of NS architectures by integrating experimental networks and enhancing NS with Machine Learning optimizations, energy-efficient slicing, and custom security features.

Authors: Alex Moura, Leobino Sampaio, Adrilene Fonseca, Arthur Urbano, Denis Menezes, Matheus Monteiro, Paulo H. M. Maia, Rafael L. Gomes

Date: May 2023

The development of new research in networking requires complex and realistic experimental environments. MicroMon is a platform designed to support the automated collection and sharing of network measurement data. NDN-Waze: A Distributed System for Vehicular Traffic Management via Named Data Networks

Authors: Guilherme Araújo, Leobino Sampaio

Date: May 2023

Many vehicular networks rely on IP-based models, which are not always secure or suitable. This paper proposes NDN-Waze, a fully distributed system for VANETs based on Named Data Networking (NDN). NDN-Waze serves as a proof-of-concept for vehicular route guidance applications. Simulation results show that NDN properties can outperform traditional IP-based data forwarding in VANETs.

Type: Conference Paper | Publication: Brazilian Symposium on Computer Networks and Distributed Systems

An Overview of Advanced Healthcare Services: Connectivity and Security in Assisted Living Systems

Authors: Adriana Viriato Ribeiro, Fernando Nakayama, Michele Nogueira, Leobino Sampaio

Date: May 2023

Assisted living environments provide advanced healthcare services, including applications in smart spaces and health monitoring regardless of location. The development of such applications involves various technologies like the Internet of Things, cloud computing, and AI. This chapter provides an overview of these services, identifying key communication and security requirements and their associated challenges.
